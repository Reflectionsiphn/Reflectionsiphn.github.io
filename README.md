# Fox Tasks

A simple, clean, and minimalist task tracker application built with vanilla JavaScript.

## Features

- Add and manage tasks effortlessly
- Mark tasks as completed
- Delete tasks
- Task counter showing remaining tasks
- Modern, responsive UI with a beautiful purple theme
- Light and intuitive design

## Tech Stack

- **HTML5** - Markup structure
- **CSS3** - Styling with Poppins font
- **JavaScript** - Task management logic
- **No dependencies** - Pure vanilla implementation

## File Structure

```
├── index.html          # Main application page
├── 404.html           # Error page for not found
├── css/
│   └── style.css      # Stylesheet
├── js/
│   └── script.js      # Task logic
└── README.md          # This file
```

## How to Use

1. Enter a task in the input field
2. Click "Add" or press Enter to add the task
3. Click the delete button on a task to remove it
4. Click on a task to mark it as completed
5. The task counter updates automatically

## Color Scheme

- **Primary Color**: #6b46c1 (Purple)
- **Background**: #f5f3ff (Light Purple)
- **Text**: #333 (Dark Gray)
- **Accents**: #e53e3e (Red for delete)
